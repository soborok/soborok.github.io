Oracle and Google APIs fight

I am studying to become a developer. Currently I am working on programming languages and later I will be working on SQL, framework, APIs and so on. Considering my current status, Google-Oracle APIs fight woudl be a difficult topic but I'd like to have a look.

This is briefly said
  Oracle has Java-related APIs and Google uses them in Android OS
  Oracle accuses that Google infringes on the APIs
  California court ruled the APIs are not copyrightable
  The appeal court concluded the APIs are copyrightable

This issue is followed by some concerns; whether Google's API use is fair-use or not, API owners could monopoize using their API.
As a novice developer I like the idea being able to use existing APIs. On the other hand, I want to be recognized and rewarded for my work.
I chose this topic for blog assignment. I thought I would have my opinion after some research. But I was running into more ideas and technology and I cannot make my opinion.
I don't have a deep insight or experience like the computer scientists but I want to consider that the purpose of the APIs and who is benefitted by having them in Android.
