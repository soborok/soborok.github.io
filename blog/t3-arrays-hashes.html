Arrays and Hashes

October 25, 2014

Variable vs Array vs Hash

In programming language, you need to declare what you are going to use. There are different kinds of things. Variable and array are one of them. Variable stores a value, only one. You can assign a value to the variable and change the value with something else.
And, array stores values, not one value but multiple values.
Let's think you are having dinner party. You invited Paul, Ted and Josh. The first guest arrived at the door is Paul. Let's assign Paul to variable 'guest1'. This string variable has one value. Second one is Ted and third one is Josh. So we will have 3 variables.

guest1 = Paul
guest2 = Ted
guest3 = Josh

These variables can be also defined using array. Remember, array can have multiple values?! Let's name the array name 'guests' and assign the values.

guests = [Paul, Ted, Josh]

You can access the values in the array by using position. In array the position starts from 0. For the first value in the array you can access like this
guest[0]
And the values is Paul and you can change the value like you do regular variable.

There is something like array, which is hash. Hash can store multiple values but it stores a pair. Let's say I love wines. Paul brought me Moscato, Ted brought me Pino Noir and Josh brought me beer. I can save this info with person's name in Hash!

guest_wine = {
Paul => Moscato,
Ted => Pinot Noir,
Josh => beer
}

We started the party and Ted suggested to have his wine first. Ok, which wine did Ted bring? uhm...aha! it was Pinot Noir. I am using 'Ted' as 'Key' and getting 'Pinot Noir' as value. A pair of key-value! It is easy and convenient, isn't it?
Now you can tell the difference between arrays and hashes. Both store multiple values but hashes remember a pair values.

Which one to use?

Uhm, then which one use?
What are you going to store? Are they paired?
Do you need keys? Do you need uniques keys other than positions?
If order is not important but it need unique keys, I will use hash.
If I am happy as long as I have the values then I will use array.
